📊 Customer Behavior Data Analytics Project
🔍 Overview
This project focuses on analyzing customer behavior using a complete data analytics workflow. It includes data loading, cleaning, exploratory data analysis (EDA), SQL-based analysis, and visualization through an interactive dashboard. The goal is to extract meaningful insights to support data-driven decision-making.

📁 Dataset
Source: Customer transactions dataset
Includes: Customer demographics, purchase behavior, product details, and ratings
Format: CSV

🛠️ Tools & Technologies
Python – Data loading, cleaning, and EDA
SQL (PostgreSQL / MySQL / SQL Server) – Querying and analysis
Power BI – Interactive dashboard creation
Gamma – Presentation (PPT) creation

⚙️ Project Steps
1. Data Loading
Imported dataset using Python (Pandas)
2. Data Cleaning
Handled missing values
Removed duplicates
Corrected data types
3. Exploratory Data Analysis (EDA)
Analyzed customer trends and patterns
Identified key metrics (sales, ratings, etc.)
Used visualizations for better understanding
4. SQL Analysis
Wrote queries to extract insights such as:
Revenue by category
Customer segmentation
Product performance
Average ratings
5. Dashboard Creation
Built an interactive Power BI dashboard with:
Sales overview
Customer insights
Product performance
6. Reporting & Presentation
Created a structured report
Developed a presentation using Gamma

📊 Dashboard
The Power BI dashboard provides:
Key KPIs (Revenue, Orders, Ratings)
Customer segmentation insights
Product performance analysis
Interactive filters for deeper exploration
📈 Results & Insights
Identified top-performing products and categories
Discovered customer purchasing patterns
Highlighted areas for business improvement
Provided actionable insights for decision-making
